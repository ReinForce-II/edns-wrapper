# EDNS Wrapper

This is a software provides CDN-friendly dns service, it uses dns.google.com as upstream.

## Install

```
git clone https://github.com/ReinForce-II/edns-wrapper.git
cd edns-wrapper
npm install
node app.js [-l <addr>] [-p <port>]
```

* Default address: 0.0.0.0
* Default Port: 3535

## Dependencies

* nodejs
* npm
